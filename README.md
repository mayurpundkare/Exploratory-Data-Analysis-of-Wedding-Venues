# 💍 WedMeGood Venue Analytics
### Exploratory Data Analysis of Wedding Venues using Python

## 📌 Project Overview

Planning a wedding involves selecting the perfect venue based on budget, capacity, ratings, and available facilities. This project analyzes wedding venue data collected from the **WedMeGood platform** to uncover meaningful business insights through **Exploratory Data Analysis (EDA)** and data visualization.

The analysis focuses on understanding venue pricing trends, customer ratings, reviews, guest capacity, room availability, and other important factors that influence venue selection.

---

## 🎯 Project Objectives

✔ Analyze venue pricing patterns

✔ Understand customer rating behavior

✔ Explore relationships between reviews and ratings

✔ Study guest capacity distribution

✔ Identify popular venue categories

✔ Discover factors affecting venue popularity

✔ Generate actionable business insights through visualizations

---

## 📊 Dataset Features

The dataset contains information related to wedding venues such as:

| Feature | Description |
|----------|------------|
| Venue Name | Name of the venue |
| Location | Venue location |
| Ratings | Customer ratings |
| Reviews | Number of customer reviews |
| Price Per Plate | Catering cost per person |
| Guest Capacity | Maximum guest accommodation |
| Rooms Available | Number of rooms |
| Venue Type | Hotel, Banquet Hall, Resort, etc. |
| Amenities | Available facilities |

## 📌 Key Insights

### 💰 Venue Pricing Trends

- Most wedding venues belong to the mid-price segment.
- Premium venues form a smaller portion of the market.

### ⭐ Customer Ratings

- Majority of venues maintain ratings above 4.0.
- Highly rated venues generally receive more customer engagement.

### 👥 Guest Capacity

- Medium-capacity venues dominate the market.
- Larger venues tend to charge higher prices.

### 📝 Reviews Analysis

- Venues with higher review counts often enjoy greater customer trust.

### 🏨 Venue Popularity

- Pricing, capacity, and facilities play significant roles in venue selection.
